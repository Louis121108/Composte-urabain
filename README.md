 Compost Urbain Autonome – Olympiades des Sciences de l’Ingénieur

##  Présentation du projet

Projet réalisé dans le cadre des Olympiades des Sciences de l’Ingénieur.

Conception d’un compost urbain autonome permettant :
- Le dépôt de déchets organiques via QR code
- La pesée automatique
- La gestion de 3 états de décomposition
- Le stockage final du compost
- La restitution du compost à l’utilisateur

Le système intègre :
- Une chaîne d’énergie
- Une chaîne d’information
- Une gestion automatisée des flux de matière
